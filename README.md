
# 국가 지적 행정 시스템 UI/UX 혁신 예시

지적재조사 등 국가 행정업무 시스템의 상담관리/지도/혁신 UI 오픈소스 예시입니다.

## 주요 기능
- 대화형(챗봇) 상담관리 UI/UX
- 지도 기반 필지 선택 및 공간 데이터 연동
- 행정 보고서 자동화 및 PL/SQL 튜닝 예시
- 레거시 환경 모던화/자동화 사례

## 폴더 구조
```
project/
├── src/
│   └── main/
│       ├── java/...
│       ├── resources/...
│       └── webapp/...
├── images/ (스크린샷, UI/UX 예시)
└── README.md
```

## 혁신 포인트 (Before/After)
- 텍스트/표 위주 → 대화형 챗봇+그래픽 지도 UI
- 수기 입력 → 지도에서 직접 필지 선택
- 반복 수작업 → 행정 보고서 자동화, 쿼리 튜닝
- Eclipse/수동 빌드 → VS Code+Maven 자동화

## 실행/구축 방법
1. Java, Maven 환경 필요
2. `mvn clean package`로 빌드
3. 샘플 DB/설정은 별도 제공

### 권한/인증/기관별 연동 안내
- 본 예시에는 실제 배포 환경에서 사용되는 권한/인증/기관별 연동(예: SSO, 내부 인증, 조직별 권한 등) 기능이 포함되어 있지 않습니다.
- 외부 PC, 테스트 환경에서는 인증/권한 관련 부분을 Mock(샘플 데이터, 임시 인증 등)으로 대체하여 사용하면 됩니다.
- "LX" 등 특정 기관명을 포함한 변수/주석/코드는 모두 일반화 또는 "기관" 등으로 변경되어 있습니다.
